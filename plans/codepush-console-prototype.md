# CodePush Console — 产品原型设计

## 一、页面结构总览

```
注册 → 登录 → Dashboard (首页)
                  ├── 服务器管理
                  │    └── 添加/编辑服务器
                  ├── App 列表 (选中服务器后)
                  │    ├── 创建 App
                  │    └── 进入 App 详情
                  │         ├── Deployment 列表
                  │         │    ├── 创建 Deployment
                  │         │    └── 进入 Deployment 详情
                  │         │         ├── Release 列表
                  │         │         │    ├── 上传 Release
                  │         │         │    └── 回滚 Release
                  │         │         └── Promote (晋升)
                  │         └── Access Key 管理
                  ├── 审计日志
                  └── 设置
                       ├── API 密钥管理
                       └── 个人信息
```

## 四、交互流程

### 4.1 首次使用流程

```
注册 → 登录 → 服务器列表（空） → 添加服务器 → 
→ 填写地址/邮箱/密码 → 测试连接 → 保存 → 
→ 自动跳转 Dashboard → 看到 App 列表
```

### 4.2 日常发布流程

```
登录 → 选择服务器 → 选择 App → 选择 Deployment →
→ 上传 Release (选择 .zip + 填写版本号 + 描述) →
→ 查看发布结果 → 审计日志自动记录
```

### 4.3 Promote 流程

```
进入 Staging Deployment → 查看 Release 列表 →
→ 点击 Promote → 选择目标 Production →
→ 确认 → 自动晋升 → 审计日志记录
```

### 4.4 回滚流程

```
进入 Deployment Release 列表 →
→ 找到要回滚到的版本 → 点击回滚 →
→ 确认 → 自动回滚 → 审计日志记录
```
